All this site web pages are written by me, and only by me, using the following computer languages:

  • XHTML5 for content;
  • CSS 3 for styling;
  • RSS 2.0 for news feed;
  • JavaScript for client-side scripting;
  • Jekyll for templating;

Every page is intended to be totally compliant with the respective published web standard. For that end, the footer of every page has logos which link to canonical markup validation engines. Please, note that the HTML5 validation always issues one warning and error since it's experimental and the use of JavaScript Web Labels is not officially standardized, respectively.

I'm a big fan of mid 90's web look and feel, and full-width article-like web pages. I've styled this site with that in mind.

No WYSIWYG editor is used. All code is written by hand using the one true editor: GNU Emacs.

This site is kindly hosted by GitHub Pages.
